Задание #5, выполнено как MilitaryPersonExtended.java
Описание задачи: реализовать предметную область - учет военного дела. Создать родительский класс "Военный состав" (фамилия, рота, звание, дата рождения, дата поступления на службу, часть) и дочерние классы:
- "Органы военного управления" (название округа, должность, выслуга лет, сумма надбавки);
- "Военная служба по контракту" (период договора, дата договора, номер протокола, сумма зарплаты);
- "Награжденные" (название награды, премия, сумма надбавки).
Реализовать класс для хранения списка военных с методом добавления нового военного и методом печати списка военных.

Задание #6, выполнено как FormulaCalculator.java
Описание задачи: Дано a, b. Создать программу, находящую min(a,b), max(a,b) и d (результат решения сложной формулы). Формула: d = (min(a,b) - 2 * sqrt(max(a,b))) / (1 + max(a,b)/min(a,b))

Задание #7, выполнено как ParameterAnalysis.java
